## 날짜: 2026-05-14

### 새로 배운 내용

파이썬의 콜렉션들에는 순서가 존재하는 리스트와 튜플, 그리고 순서가 없는 셋과 딕셔너리가 있다.
리스트, 셋, 딕셔너리는 변경이 가능한 뮤터블. 튜플은 변경 불가능한 임뮤터블

#### 리스트
- 리스트(List)란 순서가 존재하며 크기 변경이 가능한 자료구조이며 수학에서는 수열과 비슷한 개념
- 리스트는 데이터를 순차적으로 저장해 편하게 관리하기 위해 사용

#### 튜플
- 튜플(Tuple)은 리스트와 비슷하게 데이터를 순서대로 담는 자료구조이지만 리스트와 달리 데이터를 수정할 수 없다. (불변)
#### 셋
- 셋(Set)은 중복된 값을 넣을 수 없고 순서가 없는 자료구조
#### 딕셔너리
- 딕셔너리(Dictionary)는 키와 값을 한 쌍의 요소로 갖고 데이터를 수정할 수 있는, 순서가 없는 자료구조
- "Key" 핵심. Key를 꽂아야 value가 열린다.

#### 컴프리헨션
- 컴프리헨션은 “반복 가능한 객체를 순회하여 값을 선택, 변환한 결과로 새로운 컬렉션을 생성하는 파이썬의 표현식 문법”이다. (고유명사 정의)
- 가장 Pythonic하게 코딩할 수 있게 해주는 문법이 컴프리헨션이다.
- 하나의 표현식 안에 “포괄(Comprehend)” 한다.

#### 함수
- 함수란 코드를 재사용하기 위해 묶어놓은 집합

#### 고차함수
- 고차함수란 함수를 인자로 받거나 함수를 리턴하는 함수
- 고차 함수를 사용하면 코드의 재사용성이 높아지고, 간결하게 표현이 가능하며, 여러 함수를 이용해 다양한 동작이 설계 가능합니다.
