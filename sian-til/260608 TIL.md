#### 사전 훈련 모델 활용
##### model architecture
- 신경망의 전체 모델의 구조를 뜻하고 레이어의 구성, 활성화 함수, 입력 및 출력 형태 등을 의미한다.
##### 사전 훈련 모델
- 이미 훈련이 끝난 모델 또는 모델 구성요소를 뜻하고 사전 훈련 모델을 이용하면 시간과 자원을 절약할 수 있다.
##### vgg16
- 깊숙히 구조화된 CNN 구조의 사전학습 모델이고 16개의 layer로 이루어져서 vgg16이라고 명명함.
##### resnet
- 스킵 커넥션을 통해 기울기 손실 문제를 해결하기 위한 사전 훈련 모델이다.

#### 모델 튜닝 및 최적화
##### 전이 학습
- 사전 훈련된 모델을 그대로 사용하거나 추가 튜닝하는 머신 러닝 기법으로 학습시간을 단축하고 모델 성능을 개선한다. 사용 방법으로는 fine-tuning, feature extraction, zero-shot learning을 이용한다.
##### Fine tuning
- 새로운 데이터셋에 맞추어 파라미터를 조정하여 성능을 최적화 하는것.
##### 전이학습에서 Feature Extraction
- 사전 훈련 모델의 특징 추출 부분은 그대로 사용하고 분류 레이어만 새로운 데이터에 맞춰 활용하는 전이 학습 기법.
##### zero-shot learning
- 학습 데이터에 없는 새로운 데이터를 사전 정보를 활용해 새로운 학습 없이 수행하는 기법
##### 모델 비교
- 모델의 성능을 비교하여 효과적인 모델을 선택하는 과정
##### overfitting
- 학습 데이터를 과하게 학습하여 일반화 성능이 안좋아지는 것
##### underfitting
- 충분히 학습하지 못해 학습데이터와 테스트 데이터 모두에서 낮은 성능을 보이는 것.
##### 조기 중단
- 과적합을 방지하기 위해 모델의 성능이 더이상 개선되지 않는 단계에서 학습을 조기에 멈추는 정규화 기법
##### 하이퍼파라미터 튜닝
- 머신 러닝에서 사용되는 학습률이나 배치 사이즈 등 여러가지 하이퍼파라미터들의 값을 조절해가며 성능을 개선시키는 것.

#### NLP (Natural Language Processing) / 전통적 언어 모델
##### language model
- n-1개의 단어가 앞에 주어졌을때 다음 단어가 나타날 확률을 빈도수에 따라 계산하는 n-gram 언어 모델
##### n-gram 언어 모델은
- 통계 기반 언어 모델이다.
##### morphological analysis 형태소 분석
- 문장을 의미를 가진 가장 작은 단위인 형태소로 나누는 전처리를 뜻하며 한국어는 형태소 분석이 필수이다.
##### nplm (Neural Probabilistic Language Model)
- 언어 모델에 뉴럴 네트워크를 도입한 모델
##### one-hot encoding
- 범주형 값을 해당 범주 위치만 1로 하고 나머지는 0의 값을 가지는 고정길이 벡터로 표현하는 인코딩 방식 자연어를 원-핫 인코딩하면 데이터 낭비가 크기 때문에 word embedding 개념이 나옴.
##### word embedding
- 0과 1로 표현한 원-핫 인코딩과 달리 단어를 고정 차원의 “실수 벡터”로 변환하여 연속적인 값으로 표현하는 방식.
- 원-핫 인코딩에 비해 차원 수가 훨씬 적은 장점.
##### RNN (Recurrent Neural Network)
- 이전 시점의 hidden state을 다음 시점을 계산할 때 다시 사용하여 순차적인 데이터를 처리하는 순환 구조의 인공 신경망. 대체로 시계열 데이터에게 동작한다.
##### 시간 역전파 (Backpropagation Through Time)
- 층(Layer)을 역으로 가는 것이 아니라 시간을 역으로 가면서 역전파를 계산하는 것.
- CNN과 달리 층이 하나이며 이 하나의 층을 반복해서 사용한다.
##### LSTM
- RNN에 cell state 개념을 추가하여 시간성(hidden state)과 더불어 중요도(cell state)를 추가한 순환 신경망 구조이다.
