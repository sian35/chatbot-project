### 새로 배운 내용 : LangSmith
#### LangSmith
- Observation
	- 로직이 코드에 있지 않은 경우가 많다.
	- 로직을 AI가 실행한다.
	- c.f.) Logging : 로직이 모두 코드에 들어있다.

#### Tracing (추적)
- LangChain의 실행 흐름을 단계별 Run으로 기록하고, 각 단계의 입력,출력,지연시간,토큰 사용량을 트리 구조로 보여주는 추적 기능
- 체인이 실행될 때 내부에서 어떤 일이 일어났는지 확인할 수 있다.
- 원인을 빠르게 찾고 운영 안정성을 높일 수 있다.
##### Trace
- 한 번의 체인 실행 전체를 나타내는 최상위 기록
##### Run
- Trace 안에서 실행된 하나의 단계
##### Span
- Run 안에서 더 세분화된 내부 실행 단위

#### Dataset
- LangSmith에 등록된 입력.기대 출력 쌍의 모음.
- 하나의 평가 목적에 맞춰 Example을 모아 둔 묶음
- Example
	- Dataset 안의 한 평가 항목
	- inputs와 outputs로 구성
- Version
	- Dataset이 수정될 때 기록되는 변경 기록
#### Evaluator (평가자)
- 평가 가능자
	- 사람: 분야 전문가
	- AI : 해당 분야의 전문가라는 페르소나를 주고 평가하게 한다.
