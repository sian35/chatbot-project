### 새로 배운 내용
#### Optimizer
- 손실 함수를 최소화하기 위해 기울기(Gradient)를 기반으로 가중치를 업데이트 하는 최적화 알고리즘
- 경사 하강법 (Gradient Descent) 
    - 가장 대표적인 최적화 알고리즘
    - 배치 경사 하강법 (Batch Gradient Descent, BGD)
        - Batch는 전체 학습 데이터 셋을 의미함
        - **전체 데이터셋에 대해서** 에러를 구하고, 기울기를 한번만 계산한다.
        - 한번의 업데이트에 모든 학습 데이터 셋을 사용하기 때문에 시간이 많이 걸린다.
        - 메모리가 많이 필요하다.
    - 확률적 경사 하강법 (Stochastic Gradient Descent, SGD)
        - 랜덤 추출된 **한 개의 데이터**에 대해 gradient 계산 후 업데이트
        - 미니 배치 크기 1이라고 생각할 수 있다(?)
        - 불안정하게 요동치는 형태
    - 미니배치 경사 하강법 (Mini-Batch Gradient Descent, MBGD)
	    - (Mini-Batch Stochastic Gradient Descent)?
	    - 미니배치 개수만큼 가중치 업데이트가 이루어진다.
- 적응형 옵티마이저 (Adaptive Optimizer)
    - 학습률을 동적으로 조정하여 학습 효율을 높여주는 옵티마이저
    - Adagrad (Adaptive Gradient Optimizer)
        - 자주 발생하는 파라미터 업데이트에 대해 학습률을 낮추고, 드물게 발생하는 업데이트에 대해 학습률을 높인다.
        - **각 변수들에 대해 lr이 따로따로 적응하여 작용**한다.
        - $G_t$는 $lr$을 점차 줄이는 기능. $G_t$는 처음에 0으로 시작해서 $g$의 제곱을 더하면서 커진다. $g$의 값을 통해서 가중치의 변화폭, 즉 $lr$이 줄어들게 된다.
    - RMSprop (Root Mean Square)
	    - AdaGrad를 개선한 방식으로 학습률 소실 문제를 해결하기 위한 방안.
		- 학습률을 조정하면서 과거의 기울기를 반영하여 업데이트
		- EMA 지수 이동 평균을 적용한다
		- AdaGrad랑 다른 점 : $G_t$ 를 구하는 식
		- 위의 모멘텀 마지막에 사용했었던 지수평균을 사용함으로써 무작정 $G_t$ 값이 커져서 기울기가 소실되지 않고 변화량 억제의 비율은 유지시킨다.
		- $\gamma$ 에 1을 대입하면, 기존 $G_t$ 값에만 영향을 받아서, 새로운 $g_t$ (grad)에 따라 갱신되지 않고
		- $\gamma$ 에 0을 대입하면, 기존 $G_t$ 값을 제거하여, 무조건 $g_t$ 값만을 반영하게 된다.
	- Adam (Adaptive Moment Estimation)
		- Adagrad + momentum
		- combination of RMSprop & Stochastic Gradient Descent with momentum
		- 모멘텀과 RMSprop의 장점을 결합하여 학습률을 조정한다.
		- 현재 가장 널리 사용된다.
