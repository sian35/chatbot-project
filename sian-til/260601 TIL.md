### 새로 배운 내용
#### 데이터 전처리
##### 데이터 전처리
- 데이터 전처리란 데이터를 분석하고 학습에 사용하기 위해 정제하고 변환하는 과정이다.
- 모델의 성능을 향상시키고 신뢰성 높은 예측을 위해 필수적인 작업이다.

##### 데이터 증강
- 데이터 증강은 기존 데이터를 변형하여 새로운 데이터를 생성하는 것이다.
- 데이터의 양을 증가시켜 과적합을 방지하고 일반화 성능을 향상시키기 위해 필요한 작업이다.

##### 데이터셋 분할
- 데이터셋 분할은 데이터셋을 목적에 맞게 학습, 테스트, 검증 데이터 셋으로 분할하는 과정이다.
- 모델의 과적합을 방지하고 성능을 객관적으로 평가하여 일반화 성능을 확인하기 위해 필요한 작업이다.
- 종류로는 직접 분할 기법과 랜덤 분할 기법이 있다.

#### 머신러닝 기초
##### 머신러닝
- 머신러닝이란 컴퓨터 알고리즘의 분야로 데이터 속의 패턴을 예측하거나 분류하는 프로그램을 의미한다.
- 기존의 인공지능
	- 규칙 기반 시스템에 기반하여 사람이 직접 프로그래밍한 규칙에 따라 행동함
- 머신러닝
	- 스스로 패턴을 학습함.
##### Random Forest
- 의사결정 트리 기반의 앙상블 학습 기법
- 다수의 독립적인 트리를 결합하여 예측하는 모델
- 여러 트리를 독립적으로 학습하기 때문에 병렬 연산이 가능하여 빠르고 효율적으로 대량의 데이터 셋을 학습

##### K-Nearest Neighbors (K-NN)
- 새로운 데이터의 가장 가까운 K개의 데이터를 바탕으로, 새로운 데이터의 레이블 또는 예측값을 결정하는 알고리즘
- 별도의 파라미터를 추정하지 않고 전체 훈련 데이터를 저장하기 때문에 직관적이다
- 메모리 기반 학습 알고리즘 이라고 부른다.

##### Support Vector Machine (SVM)
- 주어진 데이터를 최적으로 분리하는 초평면을 찾는 지도 학습 알고리즘
- 마진을 최대화하는 초평면을 찾아 데이터 분류의 일반화 성능을 높이기 위해서 사용한다.

##### Naive Bayes
- 모든 특성이 서로 독립이라는 가정 하에 베이즈 정리를 활용하여 확률을 계산하는 분류 모델
- 서로 독립이라고 가정하기 때문에 빠르고 단순하게 분류할 수 있다.

#### 딥러닝 기초
##### 딥러닝
- 사람이 규칙을 정하지 않아도 스스로 그리고 자동으로 비선형적인 패턴을 찾아내는 기술
- 머신러닝은 알고리즘을 선택하고 훈련해야 함
- 딥러닝은 신경망 구조를 설계하는 것으로 시작하여 스스로 학습

##### Perceptron
- Neural Network의 기본 단위로 입력값을 가중치와 함께 처리하여 단일 값을 출력하는 선형 이진 분류기
- Perceptron의 기본 원리를 확장하여 multilayer perceptron 같은 복잡한 뉴럴 네트워크 구조를 사용할 수 있게 된다.

##### Activation Function
- 뉴런의 출력을 결정하는 비선형 함수
- 대표적인 종류
	- 시그모이드
	- 하이퍼볼릭 탄젠트
	- 렐루 ReLU

##### Artificial Neural Network
- 뇌의 뉴런 네트워크를 모방한 알고리즘
- 입력층, 은닉층, 출력층으로 구성

##### Fully Connected Neural Network
- 모든 뉴런이 이전 층의 모든 뉴런과 연결된 신경망 구조

##### Fully Connected Layer
- FCNN을 이루고 있는 레이어(하나의 층)를 의미함.

##### Loss Function
- 예측 값과 실제 값의 차이를 측정하는 함수
- 손실함수의 대표
	- 회귀 문제
		- MSE: 평균 제곱 오차
	- 분류 문제
		- Cross-Entropy Loss

##### Backpropagation
- 손실 함수를 통해 계산된 오차를 기반으로 각 뉴런의 weight를 조정하는 알고리즘

###### 딥러닝의 기본 원리
- 순전파를 통해 손실 함수를 사용해 오차를 계산하고
- 역전파를 통해 손실 함수에 대한 가중치의 기울기(Gradient)를 계산해 Optimizer로 가중치를 조정한다.
