### 새로 배운 내용
#### Gradient Descent
- 손실 함수를 최소화하기 위해 가중치를 반복적으로 조정하는 최적화 알고리즘
- 손실 함수의 기울기를 계산하여 가중치를 기울기의 반대 방향으로 업데이트한다.

#### Convolutional Neural Network
- 합성곱 인공 신경망
- 이미지의 공간 정보를 유지하면서 학습할 수 있는 모델
	- 이미지를 인풋으로 받았을 때 flatten할 필요가 없다.
- Feature Learning : 이미지의 특징을 추출
	- Convolutional Layer, Fully Connected Layer
- Classification : 클래스 분류
	- Fully Connected Layer

#### Convolutional Layer
- CNN에서 특징을 추출하는 레이어
- 이미지의 중요한 부분을 찾아내고, 연산량을 줄이며, 다양한 특징을 단계적으로 학습할 수 있기 때문에

#### Pooling Layer
- CNN에서 입력 특징 맵의 공간 크기를 줄여 계산량을 줄이고 중요한 특징을 추출하며 과적합을 방지하는 레이어다
- 불필요한 부분은 학습에서 제외하고 중요한 특징만 가져와서 학습
- Pooling
	- CNN에서 특성 맵의 크기를 줄이고 중요한 정보를 추려내는데 활용되는 연산

#### Flatten Layer
- 다차원 배열 형태의 데이터를 1차원 배열로 변환해주는 레이어
- 주로 FCL에 입력으로 사용할 수 있도록 하는 레이어
